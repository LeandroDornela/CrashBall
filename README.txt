INTRUCTIONS

The projet has made in the Code::Blocks IDE: http://www.codeblocks.org

To compile the code, follow the instructions:

	-Open "Source/cg.workspace" with Code::Blocks.
	-Enable c++ 11 in "Settings/Compiler/Global compiler settings/Compiler settings".
	-Add the librarie "gdi32" in "Settings/Compiler/Global compiler settings/Linker settings".
